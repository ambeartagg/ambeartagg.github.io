Music Video Rotoscoping

In this project, I created an animation for the song New Rose by the Damned. I got to use exciting new tools such as Adobe Animate and Premiere Pro, and learned about the difficult, yet fascinating process of rotoscoping. It was fun to learn about how old the technique is and how frequently it was used in early animation. It was also my first time animating anything, so the entire project was a learning experience.

Indiana Jones Animation

Since I was a child I have always loved the Indiana Jones movies, so I wanted to make a tribute piece as a project. I created an animated opening credits sequence for the film Raiders of the Lost Ark. This was only my second animation, so learning to use the tools provided in Adobe Animate was fun. I also enjoyed drawing all of the pieces in Adobe Photoshop and Illustrator and putting them together.

Coffee Shop App Design

This was my first time ever designing an application. I liked the designs created by Starbucks and similar companies so I took inspiration for my own project. In this experience I learned how to use Adobe XD and I learned more about different design principles. It was fun to make everything and put it together into a functioning blueprint.
